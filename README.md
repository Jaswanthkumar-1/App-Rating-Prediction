# App Rating Prediction (Machine Learning)

## Project Overview
This project predicts Google Play Store app ratings using machine learning. 
The goal is to identify apps that may receive high ratings based on features such as installs, reviews, category, and price.

## Dataset
Google Play Store dataset containing information about:
- App
- Category
- Rating
- Reviews
- Size
- Installs
- Price
- Content Rating
- Genres

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Steps
1. Data loading using Pandas
2. Data cleaning and preprocessing
3. Handling missing values
4. Feature engineering
5. Outlier detection
6. Exploratory Data Analysis (EDA)
7. Log transformation of skewed variables
8. Train-test split (70-30)
9. Linear Regression model building
10. Model evaluation using R² score

## Model
Linear Regression was used to predict app ratings.

## Results
The model predicts ratings based on features such as installs, reviews, and category.
